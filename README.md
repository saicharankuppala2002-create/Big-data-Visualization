# Big Data Visualization Project

## Dataset Name
Teen Mental Health Dataset

## Overview

This dataset contains information about teenagers' mental health, social media usage, sleep habits, academic performance, and demographic characteristics. The dataset is used to analyze factors affecting mental well-being among adolescents.

## Features

- Gender
- Age
- Platform Usage
- Daily Social Media Hours
- Sleep Hours
- Academic Performance
- Mental Health Score

## Project Objective

The objective of this project is to:

- Analyze social media usage patterns among teenagers.
- Study the relationship between sleep hours and mental health.
- Examine the impact of social media usage on academic performance.
- Visualize trends using charts and graphs.
- Generate meaningful insights through data analysis.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Visualizations

1. Gender Distribution (Bar Chart)
2. Social Media Platform Usage (Pie Chart)
3. Sleep Hours Distribution (Histogram)
4. Social Media Usage vs Academic Performance (Scatter Plot)
5. Correlation Heatmap

## Expected Outcomes

- Identify common social media usage patterns.
- Understand the relationship between sleep and academic performance.
- Explore factors affecting teen mental health.
- Present findings through effective data visualizations.

## Conclusion

This project provides valuable insights into the relationship between social media usage, sleep patterns, academic performance, and mental health among teenagers using data visualization techniques.
